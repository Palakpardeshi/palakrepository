Git is a version control system used by developers worldwide

Git helps you manage code history and collaborate with others

Initialize a Git repo using: git init

Stage your files using: git add .

Commit your work with: git commit -m "message"

Push to GitHub using: git push origin main

Clone a project with: git clone URL

Check the status using: git status

View the commit log with: git log

GitHub is a cloud-based hosting service for Git repositories
